百度地图Javascript API

简单介绍

百度地图JavaScript API是一套由JavaScript语言编写的应用程序接口，可帮助您在网站中构建功能丰富、交互性强的地图应用，支持PC端和移动端基于浏览器的地图应用开发，且支持HTML5特性的地图开发。

开发者文档见：JavaScript API v2.0

百度地图的使用

  • 申请密钥
    该套API免费对外开放。自v1.5版本起，需先申请密钥（ak）才可使用，接口（除发送短信功能外）无使用次数限制

  • 引入api文件

  • 创建地图实例
